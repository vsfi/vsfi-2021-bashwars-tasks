# WARRIORS!

9 tasks - 9 flags. 
Your quest is to create a one line bash script that retreives the final result.  

#### Example: 
There is a flag in some file that starts with "PEW" 
```
grep -R PEW . | cut -d':' -f2 
PEW_PEW_PEW-PEW
```
### Quick start:
```
cd bashwars1_popit
docker build . -t popit
docker run -it popit
```
